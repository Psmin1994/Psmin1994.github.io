## JWT 인증 방식 구현하기

먼저 jsonwebtoken 모듈을 설치합니다.

- 설치
  ```js
  npm i jsonwebtoken
  ```

`.env` 파일에 사용할 비밀 키를 관리합니다.

```js
// .env
ACCESS_TOKEN_SECRET_KEY = "MY_ACCESS_KEY";
REFRESH_TOKEN_SECRET_KEY = "MY_REFRESH_KEY";
```
