# 로그인 기능에 JWT를 적용해보자.

- 추가로 Crypto 모듈을 이용해 비밀번호를 암호화해보자.

---

## 비밀번호 암호화

JWT 토큰 방식 적용 전 비밀번호를 암호화해서 DB에 저장해보겠습니다.

---

## JWT 설치 및 적용

- 설치

  ```js
  npm i jsonwebtoken
  ```

`.env` 파일에서 사용할 비밀 키를 관리합니다.

```js
// .env
ACCESS_TOKEN_SECRET_KEY = "MY_ACCESS_KEY";
REFRESH_TOKEN_SECRET_KEY = "MY_REFRESH_KEY";
```
